# nors

nors is a Rust program for counting the number of rows and records in a CSV file.

## Install

```sh
cargo install nors
```

## Usage

```sh
nors <CSV file>
```
### Output Type

* plaintext (Default)
* json

```sh
nors <CSV file> -o <Output type>
```
